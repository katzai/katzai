# Katz AI Pro Chat

AI-powered chat application with location features, image generation, and more.

## Features
- User authentication with OTP
- Real-time AI chat powered by Google Gemini
- Location sharing with Telegram integration
- Text-to-speech and speech-to-text
- Image generation and analysis
- Theme customization
- Admin panel

## Deployment

This app is configured for deployment on Render.com with automatic HTTPS.

### Local Development
```bash
pip install -r requirements.txt
python app.py
```

### Production Deployment
Deployed automatically via Render.com from this GitHub repository.

## Tech Stack
- Backend: Flask (Python)
- Frontend: Vanilla JavaScript
- AI: Google Gemini API
- Styling: Custom CSS
- Hosting: Render.com
